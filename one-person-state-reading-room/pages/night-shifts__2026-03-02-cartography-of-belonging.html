Cartography of Belonging: Как картографировать нацию без территории

Ночной ресёрч by Промень · 2 марта 2026

Тезис

Карта — это не описание территории. Карта — это заявка на суверенитет. Каждая карта, которая когда-либо существовала, была политическим актом. И если network state — это нация без фиксированной территории, то ему нужна принципиально новая картография. Не карта земли, а карта принадлежности.

В этом ресёрче — пять древних традиций нетерриториального картографирования, одна критическая теория, и конкретный фреймворк для Belarus Network State 2030.

Часть 1: Карта как оружие

«Seeing Like a State»

Джеймс Скотт в своей книге Seeing Like a State (1998) показал ключевую механику: государство видит только то, что сделано legible — читаемым. Кадастровые карты не описывают существующую систему землевладения — они создают её, потому что категории карты получают силу закона.

Скотт пишет: «Государственная кадастровая карта, созданная для обозначения налогооблагаемых владельцев, не просто описывает систему землевладения — она создаёт такую систему через способность придать своим категориям силу закона».

Колониализм довёл это до предела. Доктрина terra nullius — «ничейная земля» — буквально стирала народы с карты. Если земля не была картографирована по европейским стандартам, она юридически считалась пустой. Картография XVI-XIX веков была инструментом аннексии: нарисовал карту → объявил суверенитет → стёр коренное население.

Вывод для network states: Если карта = суверенитет, то network state без собственной картографии — невидимое государство. А невидимое государство не существует в глазах мира.

Часть 2: Пять традиций нетерриториальной картографии

Но задолго до Меркатора существовали народы, которые картографировали мир иначе. Не территорию, а отношения. Не землю, а опыт. Вот пять традиций, каждая из которых содержит инсайт для network state.

2.1. Songlines (Австралия)

Аборигены Австралии создали, возможно, самую масштабную картографическую систему в истории — песенные тропы (songlines). Это устные карты, закодированные в песнях. Каждая песня описывает маршрут через ландшафт, привязанный к событиям Сновидения (Dreaming).

Ключевое: songline может пересекать территории десятков разных языковых групп. Путешественник, поющий правильную песню, получает право прохода через чужую территорию. Песня — это одновременно карта, паспорт и виза.

Антропологи фиксировали, как аборигены проходили сотни километров пустыни, ориентируясь по песням, без единого физического ориентира видимого европейскому глазу.

Инсайт для network state: Принадлежность можно кодировать в нарративе. «Песня» (shared narrative, founding myth, общая история) — это и карта, и пропуск. Для Belarus Network State — Купалле, Каляды, 1918, Пагоня — это songlines.

2.2. Stick Charts (Маршалловы острова)

Маршалловцы создали ребрёбчатые карты (stick charts) — конструкции из палочек кокосовой пальмы и ракушек. Палочки представляли не острова и не береговые линии, а паттерны океанских волн. Ракушки отмечали острова, но главное содержание карты — это поведение воды между ними.

Три типа карт:

  • Mattang — учебная, показывала общие принципы волновых паттернов
  • Meddo — секция архипелага
  • Rebbelib — полная карта цепи островов

Карты были персональными — навигатор создавал свою карту и не брал её в море. Она служила мнемоническим устройством, а не навигационным инструментом в привычном смысле.

Инсайт для network state: Карта может отображать не объекты, а потоки между ними — транзакции, коммуникации, миграционные паттерны. Для диаспоры важнее не где находятся люди, а как они связаны.

2.3. Ammassalik Maps (Гренландия)

В 1885 году инуит по имени Кунит продал датскому исследователю Густаву Хольму три деревянные тактильные карты побережья Восточной Гренландии. Вырезанные из коряги, они передавали рельеф береговой линии через форму самого объекта — карту можно было «читать» пальцами в темноте, в каяке, в шторм.

Карту можно было перевернуть, чтобы следовать побережью в обратном направлении. Одна карта — два маршрута. Размеры: 14 × 5 см и 22 × 2.5 см.

Современные исследователи установили, что карты были не навигационными, а нарративными — инструментами storytelling для передачи знаний о местности.

Инсайт для network state: Карта может быть тактильной, физической, portable. Токен, физический артефакт, wearable — объект, который является картой сообщества. NFT как Ammassalik map.

2.4. Counter-mapping (Индонезия → глобально)

Термин counter-cartography ввела Нэнси Пелузо в 1995 году, изучая лесное картографирование на Калимантане. Коренной народ даяк, столетиями управлявший лесом, начал использовать техники и технологии государственной картографии, чтобы легитимировать собственные территориальные притязания.

С тех пор counter-mapping стало глобальным движением: от Вапишан в Гайане (с помощью дронов и Digital Democracy) до палестинских маршей Tal'at, которые через коллективное движение создают деколониальную картографию.

Инсайт для network state: Используй инструменты «легитимных» государств — но для своих целей. Dashboards, censuses, on-chain attestations — это counter-mapping 21 века.

2.5. Jewish Diaspora Mental Maps

Еврейская диаспора 2000 лет существовала без территории, но с невероятно точной «картографией». Каждая синагога была ориентирована на Иерусалим. Литургический календарь привязывал к земле, на которой никто из молящихся никогда не был. Пасхальное «В следующем году — в Иерусалиме» — это одновременно навигационная точка и political statement.

Талмуд содержит детальные описания Храма и его окрестностей — по сути, textual map поддерживаемая устной традицией, по которой народ мог «ходить» по утраченной территории.

Инсайт для network state: Литургическое направление. Каждый узел diaspora network ориентирован на общий центр — даже если этот центр виртуален. Для Belarus NS: Минск как Иерусалим. Не физический доступ, а ориентация.

Часть 3: Что картографирует Network State?

Balaji Srinivasan в The Network State предложил «network archipelago» — scattered территории, сшитые цифровой инфраструктурой. Его dashboard на ns.com уже отслеживает 117 startup societies. Но все они по-прежнему привязаны к точкам на карте Меркатора.

Проблема: пин на карте — это старая картография. Она говорит «мы здесь», но не говорит «мы — это мы».

Network state нуждается в картографии, которая отображает:

Часть 4: Конкретный стек для Belarus Network State 2030

На основе пяти традиций — предложение архитектуры:

Layer 1: Census Graph (на базе EAS)

Ethereum Attestation Service позволяет создавать on-chain attestations по произвольным схемам. Для BNS 2030:


Schema: BelarusianIdentityAttestation
Fields:
  - connection_type: enum [family, friend, colleague, community]
  - shared_experience: string (e.g., "2020 protests", "Kupalle 2025 Lisbon")
  - attestation_strength: uint8 (1-10)
  - location_at_attestation: string

Каждая аттестация — это «палочка» в stick chart. Карта строится не из координат, а из подтверждённых связей.

Layer 2: Songline Protocol

Формат для encoding shared narratives:


{
  "songline_id": "1918-BNR",
  "waypoints": [
    {"event": "25 March 1918", "significance": "BNR declaration"},
    {"event": "1920 Sluцк", "significance": "armed resistance"},
    {"event": "2020 August", "significance": "peaceful revolution"},
    {"event": "2030 ???", "significance": "network state launch"}
  ],
  "carriers": ["404Embassy", "f27.club", "diaspora orgs"],
  "ritual_anchor": "25 March annually"
}

Кто знает songline — тот принадлежит. Это не paywalled content, это shared context as citizenship test.

Layer 3: Orientation Layer

Каждый узел BNS (будь то чат, физический хаб, event) имеет ориентацию на общий центр:

  • 404Embassy newsletter → каждый выпуск содержит «расстояние до Минска» — не в километрах, а в метрике сообщества (количество активных участников, новых attestations, проведённых событий)
  • f27.club → встречи начинаются с ритуала ориентации

Layer 4: Zupass-style Identity

По модели Zuzalu: zero-knowledge proof of belonging. Можешь доказать, что ты — участник BNS, не раскрывая кто именно. Или доказать, что был на Kupalle 2025, не раскрывая имя.

ZK-proof + attestation graph = карта, которую можно показать миру, не раскрывая индивидуальные координаты.

Layer 5: Physical Artifacts (Ammassalik Layer)

Физические токены принадлежности, которые являются картой:

  • Монета/жетон с encoded данными (NFC/QR → on-chain attestation)
  • Дизайн отражает граф сообщества на момент создания (generative art из connection graph)
  • Каждый артефакт уникален, потому что каждый узел графа уникален

Часть 5: Почему это важнее, чем кажется

Вернёмся к Скотту. Государства видят только то, что legible. Network state невидим для традиционных государств — у него нет кадастровой карты, нет фиксированных границ, нет территории для нанесения на карту.

Это можно воспринимать как слабость. Но пять традиций показывают другое:

  1. Songlines существуют 65,000+ лет — дольше любого государства
  2. Stick charts позволяли навигировать там, где европейские карты были бесполезны
  3. Ammassalik maps работали в условиях, где бумажные карты уничтожались
  4. Counter-mapping доказало, что картографический суверенитет можно перехватить
  5. Jewish mental maps сохранили нацию 2000 лет без территории

Network state не нуждается в legibility для внешнего наблюдателя. Он нуждается в legibility для самого себя — во внутренней картографии, которая позволяет участникам видеть друг друга, ориентироваться и навигировать.

Заключение: Первый шаг

Для Belarus Network State 2030 первым шагом должно быть не «поставить пин на карте», а:

Создать первую attestation schema и начать строить граф связей.

100 аттестаций между 50 людьми — это уже карта. 1000 аттестаций — это stick chart архипелага. 10,000 — это живая, дышащая картография нации, которая существует не в пространстве, а в связях.

Карта — это заявка на суверенитет. Пора нарисовать свою.
